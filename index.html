HTML

  • Hypertext markup language (HTML): is a language that creates the structure of a web page, like a house web develpment is started from the foundation & works its way up/out it consists of elements which work togehter to provide content and the basic frame of webpages.
  • elements: is the type of content on a webpage such as, text elements, image elements, header elements, and list elements
    • elements are also used to divide the content on the page and to describe the content to browsers/other developers common element types that divide/describe a page are meta-related elements, content divison elements, and semantic elements.
  • HTML elements cooperate to build structore of a webpage and provide a framework that can be easily navigated by the broweser to a webpage.
  • HTML elements usually consist of an opening tag and a closing tag (<,>) they contain everything that we want to include in the element.
  • each element has its own tag which are listed as follows,
  • !DOCTYPE: tells the browser the type of document listed and what to expect when opened
  • HTML represents the root of our document, inside the tag is an attribute of lang="en" which is the language of the website
  • head contains meta data (data about webpage) of web app
  • meta data tags include: title: element which is the title of the page, style: element defines the css style of the website link: element connects and external resource to the HTML document meta: element defines metadata such as character set, description, keyword, author, & viewpoint script: element embeds client-side scripts in the HTML document body: the body element contains all the contents of an HTML document like, headings, paragraphs, hyperlinks, lists etc.
